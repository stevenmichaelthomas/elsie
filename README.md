Elsie
====

Elsie is a Windows frontend to a lovely LCBO API (http://lcboapi.com/).  
A simple v1 release is slated for later this year.

## Setup ##

* `$ npm install`
* `$ grunt build`

## Running ##

* `$ cordova emulate wp8`

Deploy to a WP 8.1 phone using the Visual Studio project created by Cordova.
